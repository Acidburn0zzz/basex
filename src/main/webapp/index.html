Welcome to the BaseX HTTP Server!

You can use REST to browse your databases, switch to RESTXQ to edit web pages with XQuery, or use the filesystem to browse databases via WebDAV.

More information is found in our documentation:

  • by default, the HTTP Server provides access to the REST, RESTXQ and WebDAV services
  • the location of this page is defined by the HTTPPATH option
  • additional command-line flags can be specified when starting the HTTP server
